# Martnex Design System: Seller Dashboard

This document defines page-specific overrides and guidelines for the Seller Dashboard, extending the `MASTER.md` design system.

## 1. Aesthetic Direction: Trusted Management
The Seller Dashboard should feel like a high-end productivity tool — efficient, transparent, and precise.

- **Background**: Solid `slate-50` (Level 0).
- **Primary Surfaces**: `white` with `shadow-premium` (Level 1).
- **Active Accents**: `blue-600` (Core brand color).
- **No-Line Rule**: Avoid borders on containers. Use background tonal differences (e.g., `bg-white` card on `bg-slate-50` background).

## 2. Layout Structure
- **Sidebar**: 280px fixed width. Solid `white`. No border. Subtle `border-r border-slate-100` if necessary for clarity.
- **Header**: 80px fixed height. `bg-white/80` with `backdrop-blur`.
- **Navigation Items**:
  - `Default`: `text-slate-500`, transparent background.
  - `Active`: `bg-blue-50`, `text-blue-600`, semi-bold font.
  - `Hover`: `bg-slate-100`.

## 3. Data Presentation
- **Stats Cards**:
  - Background: `white`.
  - Icon Containers: Subtle tonal backgrounds (e.g., `bg-blue-50`).
  - Font: `Outfit` for numbers (Numerical prominence).
- **Tables**:

  - Rows: `bg-white`, alternating tonal subtle hover.
- **Charts**:
  - Primary color: `blue-600`.
  - Gradients: From `blue-600/20` to `transparent`.
